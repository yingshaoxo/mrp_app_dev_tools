从指定地址接收数据。

Syntax

int32 mrc_recvfrom(
   int32 s,
   char *buf,
   int len,
   int32 *ip,
   uint16 *port
);

Parameters

s
   [IN] 打开的socket句柄，由mrc_socket创建
buf
   [OUT] 要发送数据的缓冲区
len
   [IN] 缓存区的大小
ip
   [IN] 对端IP地址
port
   [IN] 对端端口号

Return Value

   >=0的整数 实际接收的数据字节个数
   MR_FAILED Socket已经被关闭或遇到了无法修复的错误。

Requirements

Header:mrc_base.h
