建立GPRS连接

       函数定义

              void mrc_GPRS_Connect(

GPRS_MODE mode,

gprs_event_handler handler,

DWORD timeout

);

       参数说明

              Mode      [IN] GPRS模式

              Handler   [IN] 事件句柄

              Timeout  [IN] 连接超时值，0表示使用默认值，>0则为自定义时间

       返回值

              无

       头文件

              Mrc_network.h
