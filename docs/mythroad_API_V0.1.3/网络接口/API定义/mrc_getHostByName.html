通过主机名获得该主机的IP地址值，如果一个主机的IP地址为218.18.95.203，则值为218<<24 + 18<<16 + 95<<8 + 203= 0xda125fcb。 若获取主机IP使用异步模式，使用回调函数通知引擎获取IP的结果。

Syntax

int32 mrc_getHostByName(
   const char *name,
   MR_GET_HOST_CB cb
);

Parameters

name
   p: MR_FAILED 获取IP失败; 其他值 IP地址
cb
   回调函数，见MR_GET_HOST_CB定义

Return Value

   MR_FAILED （立即感知的）失败，不再调用cb
   MR_WAITING 使用回调函数通知引擎获取IP的结果
   其他值 同步模式，立即返回的IP地址，不再调用cb

Requirements

Header:mrc_base.h
