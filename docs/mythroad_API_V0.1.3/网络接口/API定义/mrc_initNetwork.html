网络初始化，如果没有拨号，进行拨号操作。

Syntax

int32 mrc_initNetwork(
   MR_INIT_NETWORK_CB cb,
   const char *mode
);

Parameters

cb
   [IN] 当网络初始化使用异步模式时，使用该回调函数通知应用初始化结果,见MR_INIT_NETWORK_CB定义
mode
   [IN] 网络初始化拨号方式，有CMNET或CMWAP两种拨号方式。

Return Value

   MR_SUCCESS 同步模式，初始化成功，不再调用cb
   MR_FAILED （立即感知的）失败，不再调用cb
   MR_WAITING 使用回调函数通知引擎初始化结果

Remarks

  本文档描述的所有网络接口函数在实现时建议优先考虑采用非阻塞方式。若网络初始化使用异步模式，使用回调函数通知引擎初始化结果。

Requirements
