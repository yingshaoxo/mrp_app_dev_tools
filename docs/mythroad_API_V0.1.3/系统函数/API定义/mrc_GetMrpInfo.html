从当前MRP文件中读取头信息。

Syntax

int32 mrc_GetMrpInfo(
char *MrpName,
E_MRP_INFOID CMD,
uint8 *RecBuf,
int32 BufLen
);

Parameters

MrpName
[IN]MRP文件名称，含路径；
lCMD
[IN]操作命令码，值见枚举定义E_MRP_INFOID；
lRecBuf
[OUT]接收返回数据的缓冲区指针；
BufLen
[OUT]接收返回数据的缓冲区字节数

Return Value

当操作不支持、操作失败等情况时，函数返回值小于等于0；若操作成功，函数返回值为RecBuf中实际填写的字节数。

Remarks

void

Requirements

Header:mrc_exb.h
