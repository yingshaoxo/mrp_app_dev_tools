将字符串s2中的最多n个字符串复制到字符数组s1中，返回指向s1的指针。

       函数定义

              char *mrc_strncpy(char *s1, char *s2,size_t n);

       参数说明

              S1[OUT] 目标空间指针

              S2[IN]    源字符串指针

              N[IN]     字符个数

       返回值

              指向s1的指针

       备注

              如果源串长度大于n,则strncpy不复制最后的’\0’结束符，所以是不安全的，复制后需要手动添加字符串的结束符才行。
