将内存中的数据加入密钥后计算MD5。

Syntax

int32 md5MakeFromMem(
char *key,
uint8* in,
int32 offset,
int32 len,
uint8* out
);

Parameters

key
[IN]密钥。若不加入密钥，置为NULL即可。若加入密钥，则将密钥和数据一起计算MD5;
in
[IN]数据区指针;
offset
[IN]从in+offset的地址偏移开始的len个字节的数据计算MD5;如果offset<0，返回失败。
len
[IN]需要计算的数据区总字节数，若该值小于0，返回失败。如果len=0，若key!=NULL，则将返回key的MD5，
否则len==0 && key==NULL，返回的MD5值依赖于MD5函数的具体实现。
out
[OUT]若函数返回MR_SUCCESS，则通过out返回16个字节的MD5校验值,这16个字节的内存由调用者申请、释放。

Return Value

MR_SUCCESS: MD5获取成功。
MR_FAILED : 获取失败，此时out中的值是不确定的，不可作为MD5值使用。

Requirements

Header:mrc_exb.h
