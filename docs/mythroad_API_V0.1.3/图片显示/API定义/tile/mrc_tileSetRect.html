设置地砖i的可视范围。

Syntax

int32 mrc_tileSetRect(
   uint16 i,
   int16 x1,
   int16 y1,
   int16 x2,
   int16 y2
);

Parameters

i
   [IN]图片缓冲序号(地砖号)
x1
   [IN]可视范围的左上X坐标
y1
   [IN]可视范围的左上Y坐标
x2
   [IN]可视范围的右下X坐标
y2
   [IN]可视范围的右下X坐标

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Remarks

   地砖简介:地砖，背景和关卡设计的基石。图片序号与mrc_bitmapLoad中的图片序号对应，同时可以有3个地砖，图片序号0～2。

Requirements

Header:mrc_base.h
