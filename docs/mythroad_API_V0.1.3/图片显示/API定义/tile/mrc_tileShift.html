地砖i卷动

Syntax

int32 mrc_tileShift(
   uint16 i,
   uint16 mode
);

Parameters

i
   [IN]图片缓冲序号(地砖号)
mode
   [IN]0  UP；1  DOWN；2  LEFT；3  RIGHT；

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Remarks

   地砖简介:地砖，背景和关卡设计的基石。图片序号与mrc_bitmapLoad中的图片序号对应，同时可以有3个地砖，图片序号0～2。

Requirements

Header:mrc_base.h
