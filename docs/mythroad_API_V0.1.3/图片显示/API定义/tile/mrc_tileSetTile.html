设置地砖值

Syntax

int32 mrc_tileSetTile(
   uint16 i,
   uint16 x,
   uint16 y,
   uint16 v
);

Parameters

i
   [IN]图片缓冲序号(地砖号)，由mrc_bitmapNew创建
x
   [IN]地砖列
y
   [IN]地砖行
v
   [IN]地砖值

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Remarks

   地砖简介:地砖，背景和关卡设计的基石。图片序号与mrc_bitmapLoad中的图片序号对应，同时可以有3个地砖，图片序号0～2。

Requirements

Header:mrc_base.h
