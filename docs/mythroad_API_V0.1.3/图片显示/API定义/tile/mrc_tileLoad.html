从文件filename中读取地图到地砖i

Syntax

int32 mrc_tileLoad(
   uint16 i,
   char* filename
);

Parameters

i
   [IN]图片缓冲序号(地砖号)，由mrc_bitmapNew创建
file
   [IN]地图文件名

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Remarks

   地砖简介:地砖，背景和关卡设计的基石。图片序号与mrc_bitmapLoad中的图片序号对应，同时可以有3个地砖，图片序号0～2。

Requirements

Header:mrc_base.h
