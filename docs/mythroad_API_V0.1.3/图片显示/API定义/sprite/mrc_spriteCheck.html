判断碰撞的画精灵函数，返回值是画精灵时精灵所在位置 非r,g,b的象素点数目。

Syntax

int32 mrc_spriteCheck(
   uint16 i,
   uint16 spriteindex,
   int16 x,
   int16 y,
   uint8 r,
   uint8 g,
   uint8 b
);

Parameters

di
   [IN]目标图片缓冲序号
spriteindex
   [IN]精灵编号
x
   [IN]屏幕X坐标
y
   [IN]屏幕X坐标
r
   [IN]RGB R值
g
   [IN]RGB G值
b
   [IN]RGB B值

Return Value

   >=0 非r,g,b的象素点数目
   MR_FAILED 失败

Requirements

Header:mrc_base.h
