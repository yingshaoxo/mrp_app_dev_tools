Mythroad图片处理处理示例的一般流程如下：

   1、创建序号为i的bmp缓存
     ret = mrc_bitmapNew(1,bk1->width, bk1->height);

   2、装载图片
     ret = mrc_bitmapLoad(1, name, 0, 0, logo->width, logo->height, logo->width);

   3、绘制图片
     mrc_bitmapShow( 1,0,0+g_delta_y,BM_COPY,0,0,bk1->width,bk1->height);

   3、释放图片
     ret = mrc_bitmapLoad(1, "*", 0, 0, bk1->width,bk1->height,bk1->width);
