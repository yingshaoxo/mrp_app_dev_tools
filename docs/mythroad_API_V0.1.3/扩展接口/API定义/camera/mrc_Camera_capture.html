拍照

Syntax

int32 mrc_Camera_capture(
uint8 mode,
CAMERA_IMG_DATA_T* img_data
);

Parameters

mode
[IN]拍照模式，见枚举定义CAMERA_CAPTURE_MODE_T
img_data
[OUT]img_data:用于输出拍照的jpge图片数据，包括数据指针和数据大小，结构定义见CAMERA_IMG_DATA_T

Return Value

MR_SUCCESS 成功
MR_FAILED 失败
MR_IGNORE 不支持该功能

Remarks

注意:函数返回以后用户需要自己创建内存空间，将图片进行拷贝

Requirements

Header:mrc_camera.h
