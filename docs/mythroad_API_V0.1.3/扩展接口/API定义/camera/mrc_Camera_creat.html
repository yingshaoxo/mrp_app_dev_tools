创建摄像头预览界面

Syntax

int32 mrc_Camera_creat(
uint16 x,
uint16 y,
uint16 prevW,
uint16 prevH,
uint16 imgW,
uint16 imgH,
uint32 img_quality_type
);

Parameters

x
[IN]预览区域的x坐标
y
[IN]预览区域的y坐标
prevW
[IN]预览区域的宽度
prevH
[IN]预览区域的高度
imgW
[IN]拍下来的照片的宽度
imgH
[IN]拍下来的照片的高度
img_quality_type
[IN]照片质量类型，定义见枚举定义CAMERA_IMG_QUALITY_T

Return Value

MR_SUCCESS 成功
MR_FAILED 失败
MR_IGNORE 不支持该功能

Remarks

注意:prevW和prevH最大，分别为屏幕的宽和高，如果屏幕大小是240x320，那么它们的最大值分别为240和320;
imgW和imgH的最大值与摄像头的像素有关系，可以大于屏幕的宽和高。

Requirements

Header:mrc_camera.h
