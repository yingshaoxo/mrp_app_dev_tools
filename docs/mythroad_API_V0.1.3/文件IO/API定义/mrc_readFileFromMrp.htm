从mrp中读取文件的所有内容到申请的内存中。使用该函数得到的内存，需要使用mrc_freeFileData函数释放。

函数定义

              void * mrc_readFileFromMrp(

const char* filename,

int32 *filelen,

int32 lookfor

);

       参数说明

              Filename [IN]        mrp文件名

              fileLen    [OUT]     当lookfor==0时返回文件内容长度

              lookfor    [IN]        指出函数的操作形式:

0:    读取mrp中文件的内容

                    1:  仅仅查找mrp中是否存在该文件，并不读取文件内容   

       返回值

              非NULL         指向读取到的内容的指针，该内存需要调用者释放

          NULL         失败

       头文件

              Mrc_network.h
