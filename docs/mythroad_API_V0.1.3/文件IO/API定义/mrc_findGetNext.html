取得一个目录搜索的结果。将结果放置于buffer中。当目录中的结果都遍历过后，返回MR_FAILED。

Syntax

int32 mrc_findGetNext(
   int32 search_handle,
   char* buffer,
   uint32 len
);

Parameters

search_handle
   [IN]调用mr_c_findStart时返回的查找句柄
buffer
   [OUT]缓冲区，用于存放查找成功时文件名或一级子目录名
len
   [IN]缓冲区大小

Return Value

   MR_SUCCESS 搜索成功
   MR_FAILED 搜索结束或搜索失败

Requirements

Header:mrc_base.h
