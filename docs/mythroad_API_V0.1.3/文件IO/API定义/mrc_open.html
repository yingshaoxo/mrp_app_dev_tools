打开指定名称文件。

Syntax

int32 mrc_open(
   const char* filename,
   uint32 mode
);

Parameters

filename
   文件名

mode
   文件打开方式,mode取值有MR_FILE_RDONLY只读的方式打开文件;
            MR_FILE_WRONLY以只写的方式打开文件;
            MR_FILE_WRONLY以读写的方式打开文件;
            MR_FILE_RDWR以读写的方式打开文件
            MR_FILE_CREATE如果文件不存在，创建该文件
            MR_FILE_RECREATE无论文件存不存在，都重新创建该文件

Return Value

   非 NULL 文件句柄
   NULL 失败

Remarks

如果文件不存在，根据mode组合值中有MR_FILE_CREATE或MR_FILE_RECREATE则创建之。

Requirements

Header:mrc_base.h
