获得指定盘符的空间信息。

Syntax

uint64 mrc_GetDiskFreeSpaceEx(
uint8 *lpDirectoryName,
uint64 *TotalBytes,
uint64 *FreeBytes
);

Parameters

nBufferLength
[IN]lpDirectoryName，目标磁盘上的一个目录，若为NULL表示当前目录;
lTotalBytes
[OUT]该磁盘总的字节数；
lFreeBytes
[OUT]FreeBytes 该磁盘空闲字节数

Return Value

该盘符空闲字节数，当失败或空闲字节数为0时，返回0。

Remarks

目前限制当前目录必须为NULL，且必须是手机上的T卡。

Requirements

Header:mrc_exb.h
