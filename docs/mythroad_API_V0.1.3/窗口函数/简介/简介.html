Mythroad平台下常用的几种窗体形式如下：

菜单

     使用菜单时，首先调用mrc_menuCreate创建一菜单容器，然后调用mrc_menuSetItem来设置每一个菜单项，并通过 mrc_menuShow   mrc_menuRefresh来完成显示处理，退出菜单时由mrc_menuRelease 完成资源的释放；

窗体

     创建自定义窗体时，需注册两个窗口函数，一个是键盘事件回调函数PtrKeyEvent，另一个是窗体事件回调函数PtrKeyEvent。 常用的调用次序是：mrc_winInit   mrc_winNew   mrc_winClose

对话框

     对话框常做内容提示，通过按键反馈与用户进行交互，产生的按键消息为MR_DIALOG_EVENT，右软键键值为：MR_DIALOG_KEY_CANCEL，左软键键值为：MR_DIALOG_KEY_OK。 mrc_dialogCreate创建指定内容的对话框，由mrc_dialogRefresh 刷新提示内容，最后退出时由mrc_dialogRelease完成资源释放。

文本框

     文本框用来显示只读的文字信息。文本框和对话框并没有本质的区别，仅仅是显示方式上的不同，在使用上它们的主要区别是：对话框的内容一般较短，文本框的 内容一般较长，对话框一般实现为弹出式的窗口，文本框一般实现为全屏式的窗口。也可能在手机上对话框和文本框使用了相同的方式实现。
     mrc_textCreate创建指定内容的文本框，由mrc_textRefresh 刷新提示内容，最后退出时由mrc_textRelease完成资源释放。

编辑框

     编辑框用来显示并提供用户编辑文字信息，为应用提供用户输入接口，"text"是编辑框显示的初始内容。 它产生的按键消息为MR_DIALOG_EVENT，右软键键值为：MR_DIALOG_KEY_CANCEL，左软键键值为：MR_DIALOG_KEY_OK。
     mrc_editCreate创建编辑框， 由mrc_editGetText获取用户输入， 最后退出时由mrc_editRelease完成资源释放。
