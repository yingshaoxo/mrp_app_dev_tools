新建一个对话框

Syntax

int32 mrc_platDialogNew(
char* title,
char* texttext,
int32 type,
int32 textdata,
PtrHilight hOk,
PtrHilight hCancel
);

Parameters

title
对话框标题
texttext
对话框内容
type
对话框类型 MR_DIALOG_OK：对话框有"确定"键; MR_DIALOG_OK_CANCEL：对话框有"确定"和"取消"键; MR_DIALOG_CANCEL：对话框有 "取消/返回"键。
textdata
窗体数据
hOk
用户选择"确定"时的回调
hCancel
用户选择"取消/返回"时的回调

Return Value

其他 窗体句柄
MR_FAILED 失败

Requirements

Header:mrc_text.h
