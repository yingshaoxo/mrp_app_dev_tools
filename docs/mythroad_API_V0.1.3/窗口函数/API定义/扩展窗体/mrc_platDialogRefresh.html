刷新对话框

Syntax

int32 mrc_platDialogRefresh(
int32 text,
char * title,
char * texttext,
int32 type
);

Parameters

text
对话框句柄
title
对话框的标题
texttext
对话框的内容
type
对话框类型 MR_DIALOG_OK：对话框有"确定"键；MR_DIALOG_OK_CANCEL：对话框有"确定"和"取消"键；MR_DIALOG_CANCEL：对话框有 "取消/返回"键。

Return Value

MR_SUCCESS 成功
MR_FAILED 失败

Requirements

Header:mrc_text.h
