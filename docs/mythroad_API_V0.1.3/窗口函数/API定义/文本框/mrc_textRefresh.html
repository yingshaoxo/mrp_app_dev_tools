刷新文本框显示。

Syntax

int32 mrc_textRefresh(
   int32 handle,
   const char * title,
   const char * text
);

Parameters

handle
   [IN]文本框句柄，由mrc_textCreate创建
title
   [IN]文本框的标题，unicode编码，网络字节序
text
   [IN]文本框内容，unicode编码，网络字节序

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Requirements

Header:mrc_base.h
