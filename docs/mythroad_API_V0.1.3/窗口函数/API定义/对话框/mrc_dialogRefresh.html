刷新对话框的显示。

Syntax

int32 mrc_dialogRefresh(
   int32 dialog,
   const char * title,
   const char * text,
   int32 type
);

Parameters

dialog
   [IN]对话框的句柄
title
   [IN]对话框的标题，unicode编码，网络字节序
text
   [IN]对话框内容，unicode编码，网络字节序
type
   [IN]若type为-1，表示type不变,见定义MR_DIALOG_OK MR_DIALOG_OK_CANCEL MR_DIALOG_CANCEL

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Requirements

Header:mrc_base.h
