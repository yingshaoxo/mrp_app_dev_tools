在菜单里设置一个菜单项，参数index是该菜单项的序号，当用户选中某个菜单项时，把index号通过mrc_event函数发送到应用中。

Syntax

int32 mrc_menuSetItem(
   int32 menu,
   const char *text,
   int32 index,
);

Parameters

menu
   [IN]菜单的句柄，由mrc_menuCreate创建
。 text
   [IN]菜单项的名字，使用unicode编码，网络字节序
index
   [IN]菜单项的index号

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Requirements

Header:mrc_base.h
