T_MENU菜单结构体。

Syntax

typedef struct{
int32 handle;
int32 itemNum;
T_MENU_ITEM EscMenu[1];
T_MENU_ITEM subMenu[1];
}T_MENU;

Members

handle
菜单句柄，由mrc_platMenuNew创建
itemNum
菜单项的个数
EscMenu
T_MENU_ITEM结构体变量，用户不作任何选择，直接选择“返回”键时的回调函数的函数指针和参数指针。
subMenu
T_MENU_ITEM其他菜单项的回调函数指针和参数指针菜单项的个数

Requirements

Header:mrc_menu.h
