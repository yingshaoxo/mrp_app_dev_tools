T_WIN_EVENT的功能介绍

Syntax

typedef enum
{
WIN_EVENT_SHOW,
WIN_EVENT_PAUSE,
WIN_EVENT_REFRESH,
WIN_EVENT_UPDATE,
WIN_EVENT_EXIT
}T_WIN_EVENT;

Members

WIN_EVENT_SHOW
窗口创建好之后就会收到这个事件，可以在这个事件中画窗口的界面
WIN_EVENT_PAUSE
当创建一个新的窗口后，父窗口会收到这个事件
WIN_EVENT_REFRESH
当子窗口销毁掉后，父窗口会收到这个事件
WIN_EVENT_UPDATE
外部可以修改窗体中要显示的功能，然后向该窗体发送这个事件使窗体将更新的内容画到屏幕上
WIN_EVENT_EXIT
调用winDelete后，系统会向将要销毁的窗口发送这个事件，已处理窗口特殊的退出处理
