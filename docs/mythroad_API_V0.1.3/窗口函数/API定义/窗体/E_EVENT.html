事件枚举

Syntax

enum {
MR_KEY_PRESS,
MR_KEY_RELEASE,
MR_MOUSE_DOWN,
MR_MOUSE_UP,
MR_MENU_SELECT,
MR_MENU_RETURN,
MR_DIALOG_EVENT,
MR_EVENT01,
MR_EXIT_EVENT,
MR_EVENT02,
MR_LOCALUI_EVENT
};

Members

MR_KEY_PRESS   按键按下事件
MR_KEY_RELEASE   按键释放事件
MR_MOUSE_DOWN   触摸屏（鼠标）按下事件
MR_MOUSE_UP   触摸屏（鼠标）抬起/释放事件
MR_MENU_SELECT   菜单选中事件
MR_MENU_RETURN   菜单返回事件
MR_DIALOG_EVENT   对话框/编辑框/文本框事件
MR_EVENT01   VM保留，请不要使用
MR_EVENT02   应用退出事件
MR_EXIT_EVENT   VM保留，请不要使用
MR_LOCALUI_EVENT   本地化接口事件
