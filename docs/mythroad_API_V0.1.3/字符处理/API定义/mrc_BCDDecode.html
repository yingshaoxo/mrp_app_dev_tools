将压缩BCD编码的数字串解码出来。

Syntax

char *mrc_BCDDecode(
char *bcd,
int32 len,
char *output
);

Parameters

bcd
[IN]压缩BCD编码的数字串；
len
[IN]压缩BCD编码的数字串长度；
output
[OUT]输出字符串,空间由调用者负责申请和释放，本函数不负责申请释放内存。

Return Value

成功:返回指针output；失败:返回NULL

Remarks

1、BCD编码解释
2、ASCII数据转BCD见mrc_BCDEncode定义。

Requirements

Header:mrc_exb.h
