与函数mrc_drawText功能相似，不同点在于mrc_drawText将显示的字符限制在左上角为（x, y），宽高为（w, h）的矩形范围之内。

Syntax

int32 mrc_drawTextEx(
   char* pcText,
   int16 x,
   int16 y,
   mr_screenRectSt rect,
   mr_colourSt colorst,
   int flag,
   uint16 font
);

Parameters

pcTest
  [IN]待显示的字符串
x
  [IN]待显示的字符左上角x
y
  [IN]待显示的字符左上角y
rect
  [IN]矩形区域，见mr_screenRectSt定义
colorst
  [IN]字体色 RGB 值 见mr_colourSt定义
flag
  [IN] 可以取值为DRAW_TEXT_EX_IS_UNICODE   DRAW_TEXT_EX_IS_AUTO_NEWLINE，也可以是这些值的相加，即：DRAW_TEXT_EX_IS_UNICODE | DRAW_TEXT_EX_IS_AUTO_NEWLINE
font
  [IN]字体的大小 见MR_FONT_TYPE定义

Return Value

   指示出在屏幕上第一个未完整显示字符的索引，该索引是Text的unicode索引值。即使函数使用gb输入字符串，函数返回的索引值也是根据该字符串转换为
unicode串后的索引值。若所有字符都能完整显示，则返回该字串转换为unicode串后的长度。

Requirements
