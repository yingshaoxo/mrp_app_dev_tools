将gb字符串转换为Unicode字符串，并申请一片内存保存Unicode字符串，将Unicode字符串的指针返回。

Syntax

uint16* mrc_c2u(
   char *cp,
   int32 *err,
   int32 *size
);

Parameters

cp
   [IN]输入的gb字符串
err
   [OUT]填NULL；若err为非NULL，则在转换出错时err返回出错字符的索引
size
   [OUT]输出的Unicode字符串长度

Return Value

   NULL 转换出错
   其他 Unicode字符串指针

Remarks

   由于早期mythroad版本，应用使用该函数时需要附带编码转换表，使用该函数将使目标代码增大约16K。

Requirements

Header:mrc_base.h
