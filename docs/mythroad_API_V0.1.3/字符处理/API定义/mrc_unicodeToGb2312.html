将unicode编码的字符串转成gb2312编码。

Syntax

int32 mrc_unicodeToGb2312(
uint8* input,
int32 input_len,
uint8** output,
int32* output_len
);

Parameters

input
[IN]需要转换的unicode字符串;
input_len
[IN]需要转换的unicode字符串长度，字节为单位;
output
[OUT]转换成功以后的bg2312编码字符串存放缓冲区，缓冲区的内存由应用调用者提供并管理、释放;
output_len
[OUT]output缓冲区的长度，字节为单位

Return Value

MR_SUCCESS 转换成功
MR_FAILED 转换失败

Remarks

1、output缓冲区必须使用malloc分配处理的空间，否则手机可能会出现重启；
2、UNICODE编码解释
3、编码解释

Requirements
