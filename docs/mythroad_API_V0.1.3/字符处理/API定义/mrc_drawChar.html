将指定字符（仅支持unicode）绘制于屏幕指定区域，区域外的部分不显示；

Syntax

int32 mrc_drawChar(
   uint8* chr,
   int16 x,
   int16 y,
   mr_screenRectSt rect,
   mr_colourSt colorst,
   uint16 font
);

Parameters

chr
  [IN]待显示的字符unicode指针
x
  [IN]待显示的字符左上角x
y
  [IN]待显示的字符左上角y
rect
  [IN]显示的字符限制范围
colorst
  [IN] 字体色 见mr_colourSt定义
font
  [IN]字体的大小 见MR_FONT_TYPE定义

Return Value

   MR_SUCCESS 操作成功，字符为\00\00时也返回成功，但没有实质显示操作；
   MR_FAILED 操作失败，字符在字库中未找到会返回该值；

Requirements

Header:mrc_base.h
