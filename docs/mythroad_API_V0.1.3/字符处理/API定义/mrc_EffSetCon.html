将屏幕的左上角x,y，宽高为w,h的矩形区域内的r,g,b分别增强perr/256, perg/256, perb/256倍。若perr=256, perg=0, perb=0，将只保留矩形区域内的红色；若perr=perg= perb=128，将使矩形区域内产生一种半透明的效果。

Syntax

void mrc_EffSetCon(
   int16 x,
   int16 y,
   int16 w,
   int16 h,
   int16 perr,
   int16 perg,
   int16 perb
);

Parameters

x
   [IN]指定区域X坐标
y
   [IN]指定区域Y坐标
w
   [IN]指定区域宽度
h
   [IN]指定区域高度
r
   [IN]RGB R值
g
   [IN]RGB G值
b
   [IN]RGB B值

Return Value

   void
