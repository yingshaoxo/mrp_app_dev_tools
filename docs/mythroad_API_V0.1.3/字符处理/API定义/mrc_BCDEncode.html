对输入的ASCII数字串进行BCD压缩编码。

Syntax

char* mrc_BCDEncode(
char *in,
char *out
);

Parameters

in
[IN]ASCII数字串;
out
[OUT]BCD编码字符串。

Return Value

若输入参数*in和*out均不是NULL，则返回out的指针，否则返回NULL。

Remarks

1、BCD编码解释
2、输出字符串缓冲区，函数调用者负责创建该缓冲区，长度必须不少于mrc_strlen(in)/2+mrc_strlen(in)%2，此外，输出串无串结束标志。

Requirements

Header:mrc_exb.h
