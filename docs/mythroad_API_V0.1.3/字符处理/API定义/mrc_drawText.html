将指定文本绘制于屏幕指定位置。

Syntax

int32 mrc_drawText(
   char* pcText,
   int16 x,
   int16 y,
   uint8 r,
   uint8 g,
   uint8 b,
   int is_unicode,
   uint16 font
);

Parameters

pcTest
  [IN]待显示的字符串
x
  [IN]待显示的字符左上角x
y
  [IN]待显示的字符左上角y
r
  [IN]字体颜色r,g,b，r取值范围：0~255
g
  [IN]字体颜色r,g,b，g取值范围：0~255
b
  [IN]字体颜色r,g,b，b取值范围：0~255
is_unicode
  [IN] 是否是unicode，TRUE(1)表示使用unicode编码，FALSE(0)表示使用GB2312编码
font
  [IN]字体的大小 见MR_FONT_TYPE定义

Return Value

   MR_SUCCESS 操作成功，字符为\00\00时也返回成功，但没有实质显示操作；
   MR_FAILED 操作失败，字符在字库中未找到会返回该值；

Requirements

Header:mrc_base.h
