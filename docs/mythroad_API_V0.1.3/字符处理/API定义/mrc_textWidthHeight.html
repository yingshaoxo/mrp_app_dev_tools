获取字符串的显示宽度和高度。

Syntax

int mrc_textWidthHeight(
   char* pcText,
   int is_unicode,
   uint16 font,
   int32* w,
   int32* h
);

Parameters

pcText
   [IN]待显示的字符串
unicode
   [IN]是否是unicode，TRUE(1)表示使用unicode编码，FALSE(0)表示使用GB2312编码。
font
   [IN]字体大小，可能的值是MR_FONT_SMALL MR_FONT_MEDIUM MR_FONT_BIG
w
   [OUT]字符串的显示宽度
h
   [OUT]字符串的显示高度

Return Value

   MR_SUCCESS 操作成功
   MR_FAILED 操作失败

Requirements

Header:mrc_base.h
