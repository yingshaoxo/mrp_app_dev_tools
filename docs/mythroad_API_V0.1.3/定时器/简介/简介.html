利用Timer周期性触发的特点，常用它来完成状态的捕获、屏幕刷新等工作。在Mythroad平台中通过给Timer设置回调函数， 就可以在Timer触发时，自动调用其处理过程。

   在设置Timer时长时，要结合Timer回调函数里其对自身调用的位置，适当做调整，以给其它较低优先级线程运行的时间。比如：
   void UpdateScreenCallback（int32 data）
   ｛
     PaintPics();
     mrc_timerStart(GAME_LEVEL_TIMER, STONE_TIMER_INTERVAL, 1, AutoGenStoneLine,1);
   }
   即：回调后置时，Timer时应适当调小；反之
   void UpdateScreenCallback（int32 data）
   ｛
     mrc_timerStart(GAME_LEVEL_TIMER, STONE_TIMER_INTERVAL, 1, AutoGenStoneLine,1);
     PaintPics();
   }
   即：回调前置时，Timer时长应适当增大。
