代码取自“泡泡龙”游戏中炮弹装载小龙的绘制部分

   1、创建Timer
      UPDATE_LEFT_DRAGON_WORK_TIMER = mrc_timerCreate();

   2、Timer回调处理
      void LeftDragonDoWork(void)
      {
      PaintLeftDragonStatus(LEFT_DRAGON_PUSH);
      mrc_timerStart(UPDATE_LEFT_DRAGON_WORK_TIMER,500, 0, LeftDragonRecover,0);
      }

   3、启动Timer
      mrc_timerStart(UPDATE_LEFT_DRAGON_WORK_TIMER,500, 0, LeftDragonRecover,0);

   4、停止Timer
      mrc_timerStop(UPDATE_LEFT_DRAGON_WORK_TIMER);

   5、删除Timer
      mrc_timerDelete(UPDATE_LEFT_DRAGON_WORK_TIMER);
