启动定时器

Syntax

int32 mrc_timerStart (
   int32 t,
   int32 time,
   int32 data,
   mrc_timerCB f,
   int32 loop
);

Parameters

t
   [IN]定时器句柄，由mrc_timerCreate创建
time
   [IN]定时器时长，单位毫秒
data
   [IN]启动定时器时传入的data参数
f
   [IN]定时器回调函数，见mrc_timerCB定义
loop
   [IN]是否循环；0:不循环，1:循环

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Requirements

Header:mrc_base.h
