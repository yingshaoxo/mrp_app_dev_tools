停止播放声音缓冲

Syntax

int32 mrc_soundStop(
   uint16 i
);

Parameters

i
   [IN] 声音缓存序号，由mrc_soundSet创建

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Remarks

   Mythroad中最多同时支持 5个声音缓冲， index = 0~4。需要注意的是虽然Mythroad中最多同时支持5个Sound文件，但Mythroad 中这五个文件并不能同时播放，当且仅当播放的 声音类型不同并且宿主平台支持同时播放类型不同的声音时，可以同时播放不同的声音。

Requirements

Header:mrc_base.h
