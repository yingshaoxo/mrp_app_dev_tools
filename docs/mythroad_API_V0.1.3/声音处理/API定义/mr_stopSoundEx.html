停止播放声音数据

Syntax

int32 mr_stopSoundEx(int type);

Parameters

type
   [IN] 声音数据类型，见MR_SOUND_TYPE定义，此函数支持MR_SOUND_MIDI MR_SOUND_WAV MR_SOUND_MP3 

Return Value

   MR_SUCCESS 成功
   MR_FAILED 失败

Remarks

   Mythroad中最多同时支持 5个声音缓冲， index = 0~4。需要注意的是虽然Mythroad中最多同时支持5个Sound文件，但Mythroad 中这五个文件并不能同时播放，当且仅当播放的 声音类型不同并且宿主平台支持同时播放类型不同的声音时，可以同时播放不同的声音。

Requirements

Header:mrc_base.h
