Mythroad声音播放处理示例

   1、添加语音数据
     typedef struct {
     int32 len;
     uint8 type;
     void* pData;
     } T_SOUND_MAP;//声音说明结构本
     static T_SOUND_MAP sound_map[GAME_SOUND_MAX_ID];//声音缓存组
     mrc_readFileFromMrpEx(NULL,name,(uint8**)&(sound_map[id].pData),(int32 *)&(sound_map[id].len),0);//从MRP资源包中装载声音数据

   2、播放语音数据
     mr_playSoundEx(sound_map[index].type, sound_map[index].pData, sound_map[index].len, loop);

   3、停止播放语音数据
     mr_stopSoundEx(sound_map[index].type);
